lumos vr : A virtual Reality Gaming Experience

Duration : two weeks

Rescue fireflies from a nearby gas leak threat. Catch as many as you can before the place is fully covered in smoke ( 2 minutes 22 seconds) and release them in your safe front yard. The more you save, the happier you'll feel. Experience a beautiful visual reward upon their release.

Ideation Phase

Rescue as many fireflies as you can within a minute. Navigate swiftly, catch fireflies before the gas harms them, and release them to safety in your backyard. Aim for a high rescue count to unlock a stunning visual reward upon their release.

core Loops

Firefly Rescue VR is an immersive game where players explore a VR forest to collect fireflies before smoke engulfs the environment, acting as a timer. Using VR controllers or gestures, players navigate and capture fireflies, earning points with each capture. Immediate feedback through visuals and sounds enhances engagement, while reaching score milestones unlocks rewards like a stunning display of fireflies dancing around the player, creating a magical experience. The game features adjustable difficulty levels, dynamic smoke progression, and interactive forest elements to keep the gameplay challenging and rewarding.

process review

Installation of XR Interaction Toolkit:

We installed XR Interaction Toolkit in Unity to enable VR Interactions.

Configuration of VR Settings:

We configured necessary settings in Unity for VR experiences.

Asset Selection from Storyboard:

Assets from the storyboard were selected upon and chosen further for the game.

VR Intro Creation:

Using Unity’s Asset Store and blender, the cave scene was created.

Terrain Creation:

Using Unity’s Terrain System, the terrain was created.

Asset Acquisition and Modification:

Obtain tree and grass samples from Unity Asset Store and modify them according to project needs.

Game Script Creation

Game script was developed to facilitate interactions and gameplay mechanics.

Implementation of Sound Design:

Narration - AI text to speech software (Resemble AI) Ambient sound - Veritasium Background Music (Pixabay) Special effects - Firefly sound - Magic sound effect (Pixabay)

First Prototype Implementation:

  • Implement basic gameplay elements:
  • Implement grabbing functionality.
  • Spawn enemies within the environment.
  • Establish kill zones with necessary colliders.
  • Integrate scoring system and timer.
  • Develop a game controller with three states. (Start, Waiting and Over)

Second Prototype Implementation:

  • Game Play mechanics and features were further improved.
  • Improve physics of the firefly for more realistic behavior.
  • Added the treasure chest and the final level.
  • Implement haptic feedback for enhanced user experience.
  • Utilize fog as a diegetic representation for the timer.
  • Enhance lighting within the environment.
  • Add post-processing effects to improve visuals and atmosphere.

Iterative Testing and Refinement:

Conducted user testing to gather feedback and improve on the game.


Future Enhancements:

  • Adding different elements to make the game more challenging for the users and making Level Designs.
  • Creating 3D ambient sound for more immersive experience
